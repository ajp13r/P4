Room Status Detector

A look at my IoT project.

Project Overview

IoT device that detects whether a room is occupied or not using a sonic sensor and an Arduino microcontroller. It is connected to discord api and sends a message to a discord channel whenever the room status changes.

Technologies Used
Breadboard Arduino Sonic Sensor Discord API
What I Learned

This project taught me how to work with IoT devices and sensors, as well as how to use the Arduino platform for prototyping. I also gained experience with integrating hardware with software by connecting the Arduino to the Discord API. Overall, this project helped me develop my skills in both hardware and software development, and gave me a better understanding of how IoT devices can be used in real-world applications.
