VR Cooking Simulator

A look at my game development project.

Project Overview

A virtual reality cooking simulator that allows users to practice cooking skills in a safe, immersive environment. The game features realistic cooking mechanics, a variety of recipes, and a fun, engaging atmosphere.

Technologies Used
Unity C# Meta Quest 3 Blender
What I Learned

This project taught me how to design and develop a VR game using Unity and C#. I learned about VR mechanics, user experience design, and 3D modeling. I also gained experience with the Meta Quest 3 platform and its capabilities. Overall, this project helped me develop my skills in game development and gave me a deeper understanding of VR technology.
